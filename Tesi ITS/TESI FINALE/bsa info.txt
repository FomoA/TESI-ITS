Ambienti di test
Nella tabella seguente sono elencati i collegamenti alle API (attraverso intefaccia Swagger) nei vari ambienti di test.



Swagger è un linguaggio di descrizione dell'interfaccia per descrivere le API RESTful espresse utilizzando JSON.


L'infrastruttura è così strutturata:

in test c’è un una macchina in green zone ed una in DMZ (no vip quindi) e occorre chiamare direttamente il server http://ls001s01-00-api.rmasede.grma.net:8888
in prep (3 nodi in DMZ)
esiste un vip raggiungibile da dentro nostra intranet https://vipprep-apiext.rmasede.grma.net/gateway (certificato selSigned)
l’indirizzo ufficiale da Internet é https://api-gateway-pre.grupporealemutua.it/gateway (certificato rilasciato da CA ufficiale, GlobalSign)
